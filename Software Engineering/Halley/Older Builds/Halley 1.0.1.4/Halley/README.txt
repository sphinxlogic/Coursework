TO Build "Halley"

1. Open a Visual Studio 2008 command prompt and navigate the the Halley Directory.

2. Run this command on the command Line "msbuild build.proj"

you might get some warnings but don't 'worry about those right now.

Most of the warnings are XML comments which are easy to add.

The Visual Studio 2008 Command Prompt is located at Start->All Programs->Microsoft Visual Studio 2008->Visual Studio Tools.
